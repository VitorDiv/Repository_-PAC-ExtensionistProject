Estrutura condicional

O que são condicionais?

Condicional em programação é como uma bifurcação em uma estrada. Quando você chega a uma interseção, precisa decidir para onde ir, certo? Imagine que você está dirigindo e chega a uma bifurcação na estrada.

  • Se você quer ir para a esquerda, você vai para a esquerda;
  • Se não, ir para a direita.

Na linguagem C, usamos essas condicionais, como if (se) e else (senão), para fazer o programa tomar decisões com base em certas condições. Por exemplo:

  • Se (if) estiver chovendo, você pode decidir levar um guarda-chuva (condição verdadeira);
  • Se não (else), você não precisa (condição falsa).

Condicional é basicamente isso: fazer escolhas com base em situações diferentes. Pode ser algo tão simples quanto decidir o que vestir com base no clima, ou algo mais complexo, como decidir qual caminho seguir em um jogo de aventura. Então, condicionais são como direções que um programa segue, dependendo das condições que são estabelecidas, assim como nós tomamos decisões na vida cotidiana.
