Bibliotecas básicas em C

O que são as bibliotecas em C?

As bibliotecas padrão são como coleções de ferramentas úteis que os programadores podem usar em seus programas escritos em linguagens de programação, um exemplo é a linguagem C. Cada biblioteca tem um conjunto de funções que podem ser chamadas para realizar diferentes tarefas, como manipular texto, fazer cálculos matemáticos, lidar com o tempo e muito mais.

Para acessar essas bibliotecas, os programadores precisam incluir uma cópia do arquivo de interface correspondente em seus programas. Por exemplo, se você quiser usar funções da biblioteca "stdlib", você incluiria o arquivo "stdlib.h" em seu programa. Veja algumas amostras enfeitadas de interfaces das principais bibliotecas padrão C:

  • stdlib.h: Implementa funções para diversas operações, incluindo conversão, alocação de memória, controle de processo, funções de busca e ordenação.

  • stdio.h: Nesta biblioteca há funções úteis para entrada e saída de dados (std = standard, padrão em inglês; io = Input/Output, entrada e saída ==> stdio = Entrada e saída padronizadas).

  • math.h: Ela permite usar funções matemáticas básicas, tais como potência, trigonométricas, exponenciais, logarítmos, etc.

  • string.h: Contém uma série de funções para manipular strings.

  • limits.h: Esta biblioteca define constantes que especificam os tamanhos máximo e mínimo de vários tipos-de-dados básicos no seu computador.
